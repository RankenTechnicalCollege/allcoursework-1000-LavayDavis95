Network Protocols

HTML
Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser.
HTTPS
is an extension of the Hypertext Transfer Protocol (HTTP). It uses encryption for secure communication over a computer network, and is widely used on the Internet.
DNS
is a hierarchical and distributed name service that provides a naming system for computers, services, and other resources on the Internet or other Internet Protocol (IP) networks.
